## Overview

A small React app that lets users search for a city and view current weather conditions.

## Features

- Search by city name
- Displays temperature, humidity, and wind speed
- Responsive layout for mobile and desktop

## Tech Stack

- React + TypeScript
- OpenWeatherMap API
- CSS Modules
